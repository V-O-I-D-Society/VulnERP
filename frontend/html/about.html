Welcome to VulnERP

A deliberately vulnerable college ERP environment for practical security testing and learning.

Platform Objective

VulnERP helps learners identify real-world security weaknesses commonly found in enterprise applications, particularly in authentication and user input handling.

Previous Screen Explanation

The initial login page relied entirely on client-side logic. This was intentional and demonstrates why frontend-based security controls should never be trusted.

How Challenges Work

Not all vulnerabilities appear after login. Some issues may exist earlier in the workflow, even before role-based access is granted.

Note: This is not a game that always begins after clicking a start button. Observe how data is processed on each page — sometimes the vulnerability is already in front of you.
This application is provided strictly for educational and ethical security practice. Testing real systems without authorization is illegal.
